# TrafficRiskDetection
It is a deep learning and computer vision project using latest libraries like YOLO 26 for object detection etc and analysing if two objects are at a risk of an accident.

Dataset:
https://www.kaggle.com/datasets/arunavfc11/indian-traffic-videos?select=13105470_3840_2160_30fps.mp4
https://www.kaggle.com/datasets/siddhi17/road-crossing-dataset
